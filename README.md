The Bangladesh Bank – Console Banking System
Description

This is a simple console-based banking system implemented in C programming language. The program allows multiple users to:

Check their account balance

Deposit money

Withdraw money

Exit the program

It uses arrays to manage multiple accounts and ensures basic validations for deposits, withdrawals, and account selection.

Features

Multiple Accounts Support

The program can handle 5 accounts simultaneously.

Each account has its own balance stored in an array.

Menu-Driven Interface

Users interact with the program through a simple menu system.

Options include:

Check Balance

Deposit

Withdraw

Exit

Validations

Prevents deposits or withdrawals of negative or zero amounts.

Ensures withdrawals do not exceed account balance.

Checks for valid account numbers (1–5).

How to Use

Run the program.

Enter an account number (1–5).

Choose an option from the menu.

For deposit or withdrawal, enter the amount.

Check balance anytime to see the updated balance.

Select Exit to close the program.
